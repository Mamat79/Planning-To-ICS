# Limites connues

- L'application crée un fichier ICS ; l'ajout dans Outlook ou un autre agenda reste une étape manuelle.
- Le bouton de mise à jour vérifie la dernière Release GitHub mais n'installe pas silencieusement une nouvelle version.
- Le paquet macOS n'est pas signé ni notarisé par Apple. Au premier lancement, utiliser clic droit puis **Ouvrir**.
- Un PDF scanné sans couche texte peut être signalé comme non reconnu ; il faut alors un PDF exporté avec du texte exploitable.
- Un import manuel ne garantit pas que tous les logiciels de calendrier supprimeront automatiquement un événement retiré d'un nouvel export.
